DuckDNS Setup Instructions

Enable Cloudflare DDNS Automatically keep a specific Cloudflare 'A' record updated with your public IP.
Cloudflare Setup Instructions
  1. Login to Cloudflare Dashboard and select your top-level domain.
  2. Create a new 'A' record pointing to your server (e.g. server.mydomain.com). Make sure it exists!
  3. Locate your Zone Identifier on the Overview page of your domain (bottom right).
  4. Go to My Profile > API Tokens.
  5. Create a Scoped Token using the "Edit zone DNS" template, restricting it to your specific zone. Note: Global API Keys are insecure and not recommended for this application.
