Use an Existing Folder

Use this when your storage is already set up, such as ZFS, RAID, mergerfs, SnapRAID, NFS, or a normal folder. SimpleSaferServer will not mount or format anything.

Storage Marker

SimpleSaferServer creates a small hidden folder named .simple-safer-server inside your storage location. The marker file helps stop cloud backup from syncing an empty or wrong folder if storage is not mounted correctly.
